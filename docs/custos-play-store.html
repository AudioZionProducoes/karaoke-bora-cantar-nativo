Detalhamento

1. Taxa de Desenvolvedor Google ($25)

  • Paga uma unica vez ao criar a conta
  • Valida para sempre - nunca expira
  • Permite publicar quantos apps quiser
  • Pago diretamente ao Google (nao ao Replit)

2. Atualizacoes do App (Updates)

  • Totalmente GRATIS
  • Pode enviar quantas atualizacoes quiser
  • Google revisa cada update (pode levar horas a dias)
  • Usuarios recebem automaticamente via Play Store

3. Musicas no Bunny.net

VOCE NAO PRECISA ATUALIZAR O APP!

  • O app mobile le as musicas da sua API backend
  • Sua API le as musicas do Bunny.net
  • Quando voce atualiza musicas no Bunny.net, elas aparecem automaticamente no app
  • O app nao precisa de update para ver musicas novas

4. Taxa sobre Vendas (se aplicavel)

So se voce vender assinaturas ou pacotes DENTRO do app:

  • Google cobra 15% nos primeiros $1M/ano
  • Depois 30% sobre o excedente

Para negocios B2B (bares pagam voce diretamente via WooCommerce, fora do app): - Nao paga taxa ao Google - O app e so uma ferramenta para os clientes usarem
